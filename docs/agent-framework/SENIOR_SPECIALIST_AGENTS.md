# Senior Specialist Agents

## Overview

Specialist agents are domain experts that execute gates assigned by the Orchestrator. Each specialist maintains **6–8 maximum active skills** to prevent context overload.

---

## Senior Engineering Agent

**Domain:** Next.js API routes, server modules, Prisma/database, authentication, infrastructure, integrations

**Active Skills (max 8):**

1. `api-contract` — typed contracts between Next.js routes, server modules, and UI
2. `data-validation` — Zod/Prisma-safe validation patterns
3. `input-sanitisation` — XSS, SQL injection, command injection prevention
4. `structured-logging` — JSON logging with correlation IDs
5. `error-taxonomy` — error codes, categories, user-facing messages
6. `state-machine` — finite state machine patterns for complex flows
7. `resilience-patterns` — retry, circuit breaker, graceful degradation
8. `evidence-verifier` — verify API endpoint responses are valid artefacts

**Output format:** Code changes + curl proof + test results

## Senior LMS Content Agent

**Domain:** Course structure, IICRC compliance, learning pathways, CECs

**Active Skills (max 6):**

1. `outcome-translator` — translate "course ready" into content checklist
2. `definition-of-done-builder` — create measurable done criteria for course content
3. `blueprint-first` — wireframe course structure before building
4. `evidence-verifier` — verify IICRC compliance artefacts
5. `finished-audit` — LMS-specific production readiness check
6. `structured-logging` — track content completion and gaps

**IICRC Compliance Rules:**

- Existing 40 courses: IICRC CEC-approved (board voted, display allowed)
- New courses: must be submitted to IICRC America before claiming CECs
- Correct terminology: "CECs" (Continuing Education Credits), not "CPD hours"

**Output format:** Course structure + compliance checklist + CEC count
